## 五．证明题

1. 设函数 $f(x)$ 在 $( - \infty, + \infty)$ 上有定义，对任意的 $x,y \in ( - \infty, + \infty)$ 有 $f\left( x + y \right) = {\ f}\left( x \right) + f\left( y \right) + xy$，且 $f'(0) = 1$，证明 $f'\left( x \right) = 1 + x$.

   (2) 又设 $f'\left( 0 \right) = a$ (常数)， 证明 $f\left( x \right) = ax$.
